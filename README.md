## Project Overview
A SQL database for tracking gaming platform data including players, games, achievements, and purchases.

## Database Features
- 7 normalized tables with relationships
- Sample data for 100+ virtual players
- Business analytics queries
- Player segmentation and revenue analysis

## Tables Created
1. `players` - Player profiles and spending
2. `games` - Game catalog with pricing
3. `player_games` - Ownership and playtime
4. `achievements` - In-game achievements
5. `player_achievements` - Achievement unlocks
6. `in_game_purchases` - Microtransactions
7. `friends` - Player social network

## How to Use
1. Run `schema.sql` to create tables
2. Run `sample_data.sql` to add data
3. Run `queries.sql` for analytics
